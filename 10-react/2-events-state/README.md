Состояние компонентов
===
## Выполнив это задание, вы:

- научитесь создавать и управлять состоянием в классовых компонентах с помощью `this.state` и `this.setState()`;
- поймёте, как организовывать взаимодействие между компонентами, передавая функции-обработчики через props (принцип «поднятия состояния»);
- научитесь фильтровать и отображать данные в зависимости от состояния компонента;
- попрактикуетесь в условном рендеринге для переключения между различными вариантами отображения интерфейса.

Состояние (state) — это сердце любого интерактивного React-приложения. Это задание научит вас использовать состояние для создания интерактивных интерфейсов. Вы увидите, как по клику пользователя фильтруются списки или меняется вид отображения элементов. Умение управлять состоянием — это важный навык, необходимый для разработки приложений на React, где интерфейс динамически обновляется в ответ на ввод пользователя.

## Описание задания

Необходимо выполнить и предоставить на проверку обязательную задачу. Вторая задача выполняется по желанию:

1. [Портфолио с фильтрами](filter) — обязательная задача.
2. [Расположение товаров](layouts) — необязательная задача.    

## Инструкция к выполнению

Если вы будете выполнять обе задачи, то их лучше сдавать в разных репозиториях, то есть через vite реализовать два проекта, чтобы не
было конфликта стилей. Но если вы позаботитесь о том, что конфликта не будет, то можете сдавать и в одном проекте.

Все стили необходимо размещать в файле App.css.

Документация по созданию приложения [React](https://vitejs.dev/guide/).

1. Откройте терминал и пропишите следующую команду: `yarn create vite my-app --template react`,
   либо `yarn create vite my-app --template react-ts`, если
   нужен шаблон с TypeScript. Эта команда создаст настроенный
   шаблонный проект.
2. Откройте созданный проект в своей IDE.
3. Установите зависимости.
4. Готово. Чтобы запустить приложение, введите команду: `yarn dev`(либо `npm run dev`).
