Props
====

Необходимо выполнить и предоставить на проверку следующие задачи:

1. [Список предложений](listing).
2. [История сообщений в чате](chat) — необязательная задача.

Все задачи лучше сдавать в разных репозиториях, то есть реализовать два проекта, чтобы не
было конфликта стилей. Но если вы позаботитесь о том, что конфликта не будет, то можете сдавать и в одном проекте.

**Важно!**  
Задачи должны быть написаны на TypeScript. Все Props должны быть типизированы. Использование типа any не
допускается.
